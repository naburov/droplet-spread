### Accelerated onset (`fast_*` configs)

Generate with:

```bash
python scripts/generate_chainsaw_ab_configs.py --fast-only
```

Changes vs standard `ref`:
- Wang-like: CA=120°, U∞=1.5, We₂=4.5, r=0.12, slip=0.02
- Larger timestep cap: `dt=1e-3`, CFL coefficients 0.35
- `smooth_curvature: false` (except `fast_wall_energy`)
- `t_max≈0.055` (~few k steps to see strip95 blow-up on `fast_analytic_fullwall`)

Compare `fast_analytic_fullwall` vs `fast_wall_energy` first.

Enable runtime CSV via `solver_params.chainsaw_diagnostics: true` (set in generated configs).
